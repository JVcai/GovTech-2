# 🌾 AI-Terminal: Subsidy Distribution System (GovTech)

**AI-Terminal** — это интеллектуальная система поддержки принятия решений (DSS) для Министерства сельского хозяйства, разработанная для меритократического распределения субсидий.

### 🚀 Key Features:
- **ML-Scoring:** Анализ заявок с помощью XGBoost на основе ROI, стажа и социальных векторов.
- **Explainable AI (XAI):** Использование матриц SHAP для визуализации факторов влияния.
- **LLM Agent (Gemini 2.5 Flash):** Умный чат-интерфейс для фильтрации данных и генерация управленческих резюме.
- **Human-in-the-loop:** Возможность ручной корректировки решений с аудиторским следом.

### 🛠 Tech Stack:
- **Frontend:** Streamlit (Custom CSS)
- **Backend:** Python (Pandas, NumPy)
- **AI/ML:** XGBoost, SHAP, Google Gemini API
